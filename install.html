Maple: Your first steps

But, first

Before you begin, I highly recommend you make sure you understand the below things

Unofficial Software

Maple itself is not a supported Apple software. While Maple has undergone testing, Apple does not recommend you use software like this. Any Leaves you may install are even further unsupported and unofficial, and there are no liabilities.

See Maple's Security Guidance

Security Concerns

Installing Maple requires that you disable System Integrity Protection on your Mac. Disabling this software prevents your Mac from being able to stop unauthorized code from running.

Learn More about SIP 

Data Loss

Maple itself will not cause any data loss. However, it does enable developers and yourself to modify the system however they would like. Without knowing what you're doing, this could cause data loss.

See Maple's Security Guidance

Make Sure You're Supported!

Maple Supports the Following MacOS Versions: 

MacOS 11.0 Big Sur or Greater

MacOS 12.0 Monterey or Greater

Support for MacOS Ventura is hopefully on the way, but may take time as extra safeguards have been implemented in the new operating system.

Ready? Just follow the below steps

Download Maple from the Official Source

Only download Maple from the official Github releases page. The download button below will download the .DMG file directly from The Official Maple Github

Download Maple 

Move Maple To Your Applications Folder

Once downloaded, open the .dmg file, and then drag Maple into the Applications Folder Shortcut. Enter your password when prompted.

Open Maple, and follow the install directions

Maple will ask for your Admin password up to two times. When needed, please enter this password to allow Maple to run privileged code. For concerns about this, see Maple's Security Guidelines.

NOTE: If your Mac says that it cannot be opened for one reason or another, open System Preferences, and go to Security & Privacy. Under the general tab, click "Open Anyway" and then "Open" in the popup. You should only have to do this once.

Check to make sure Maple is configured properly

1. Open Maple's Status Bar app, by clicking on the symbol at the top of your screen
2. Hover over the ellipsis, and select Settings
3. In the first panel, ensure there are no red labels. If they are, address them below

You're all set! Begin installing Leaves and customizing your Mac

SIP is enabled: Maple cannot inject code

SIP, or System Integrity Protection, is a safehold which MacOS implements to attempt to prevent unauthorized or dangerous code from running on your machine. While this program is not a guarantee of preventing malware, it is a guarantee of preventing Maple from working. This is because Maple changes the authorized code signature of most applications on your device, and with SIP enabled, will prevent these programs from launching. Due to this, Maple has a safehold to ensure that SIP is disabled before injecting code. 

Please follow the directions on Apple's Official Website to disable SIP. As it will require a reboot, I suggest opening this page on your smartphone instead.
