Security

Today, you won't do that

Maple utilizes security faults to work, so it's safe to assume your device is still vulnerable. However, there are ways you can still protect your device, even while using it. 

SIP

SIP was developed by Apple to prevent unauthorized code from running. Disabling it could allow malicious apps to run.

Learn More on Apple's website 

Developer Trust

Developers build the leaves. Maple is a powerful application, and to reflect that, it's been opensourced, in order to show that it's safe. When installing Leaves, make sure that you trust the developer and the code that's running.

Staying Alert

It's important when running arbitrary software to make sure that you stay aware to any queues that could show you malicious software is being run. Watch out for suspicious network traffic, missing or extra files, slower than usual performance, or newly spawned processes.
