MapleKit

The all-in-one reference with everything you'll need

MapleKit was developed by me, and empowers you to connect to Maple. User preferences and Logging is the focus, with a goal to expand to more areas.

Checkout the GitHub and Docs 

Orion

The backbone of what runs Maple, and powers you to get your code where you want it.

Orion was developed by the same team that brought Theos and Orion itself to the Jailbreak community. Now, it powers you to customize your Mac however you'd like.

Check out the GitHub and Docs

Maple Was Built For Developers

This is why we created it

Maple was born from a desire to make customizing the Mac easier for the developer. While other tools do exist, most are poorly documented, and have few or no examples. Maple is supposed to be different. Can it improve? Help me make Maple better!
